Project Overview
This repository contains a specialized implementation of Vision Transformers (ViTs) designed for high-energy physics analysis within the CMS End-to-End (E2E) framework. The project addresses the computational bottleneck of processing high-resolution 125x125 detector images by replacing standard quadratic self-attention with linear-scaling alternatives. The model is engineered to perform simultaneous multi-task learning, specifically targeting the classification of particle jets and the precision reconstruction of their invariant mass.
Technical Implementation
The architecture leverages two primary pathways for efficiency: Linear Attention and Performer mechanisms. By utilizing the associativity property of matrix multiplication and Orthogonal Random Features (ORF), the pipeline achieves O(N) complexity, enabling the model to capture global spatial correlations across 8-channel detector data without the memory constraints of traditional transformers. The training workflow includes a self-supervised pretraining phase on large-scale unlabeled datasets to establish robust feature priors, followed by fine-tuning on labeled samples with a dual-head output for classification and regression.
Performance and Metrics
Evaluation is focused on physical accuracy and computational efficiency. Classification performance is measured using Area Under the ROC Curve (ROC-AUC), while the regression head is assessed through Root Mean Squared Error (RMSE) and mass resolution. The pipeline includes optimized data loaders for handling large-scale HDF5 files and utilizes mixed-precision training to accelerate convergence. This project serves as a benchmark for deploying efficient transformer architectures in resource-intensive scientific environments.
